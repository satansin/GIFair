# learning-fair-representations
We mainly follow the python implementation of Zemel et al. 2013
http://www.cs.toronto.edu/~toni/Papers/icml-final.pdf

Dependencies:

$ conda create --name gifair_lfr python=3.8
$ conda activate gifair_lfr

$ conda install -c anaconda scipy
$ conda install -c conda-forge numba=0.48.0
$ conda install -c conda-forge scikit-learn
$ conda install -c conda-forge pandas
